# 履歴書メーカー

## 開発したきっかけ
Flaskを使ったwebアプリ開発の練習

## アプリの概要
名前、生年月日、高校名、大学名、浪人年数を入力すると、現在の年齢・入学年度・卒業年度が記載された履歴書の形で表示される。
